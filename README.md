# SQL Injection Training (모의 해킹 체험용)

이 프로젝트는 보안 체험 부스 및 교육을 위해 제작된 **단계별 SQL 인젝션 학습용 웹 애플리케이션**입니다. 사용자는 실제 취약한 환경에서 공격을 시도하고, 보안 설정(Prepared Statement)에 따른 차이를 체험할 수 있습니다.

## 🛠 주요 기능 및 미션 구성

* **단계별 미션 시스템**: 총 6단계의 미션으로 구성되어 있으며, 각 단계를 클리어해야 다음 단계로 진행이 가능합니다.
    * **Level 1**: 로그인 우회 (OR 연산자 등을 이용한 관리자 로그인)
    * **Level 2**: 숨겨진 게시글 열람
    * **Level 3**: 데이터베이스 구조 유출 (UNION Injection을 이용한 테이블 명세 확인)
    * **Level 4**: 사용자 비밀번호 탈취
    * **Level 5**: 시스템 플래그(Flag) 획득
    * **Level 6**: 게시글 임의 추가
* **보안 모드 토글 (Secure Mode)**: 'OFF(취약한 쿼리)'와 'ON(Prepared Statement 적용)' 모드를 실시간으로 전환하여 방어 원리를 학습할 수 있습니다.
* **자동 데이터베이스 초기화**: `db/db_init.php` 실행 시 `user_info`, `posts`, `flags`, `top_secret` 테이블과 초기 실습 데이터가 자동으로 생성됩니다.

##  설치 및 주의 사항

1.  **DB 설정**: `db/db_init.php` 파일에서 실제 MySQL 연결 정보(`host`, `dbname`, `user`, `password`)를 설정하십시오.
2.  **초기화 규칙**: 서버 업로드 시 `db_initialized.txt` 파일은 포함하지 않아야 정상적으로 초기화 로직이 작동합니다.
3.  **UI 패치**: 모든 모달 창은 엔터 키로 닫을 수 있으며, 정답 제출 시 비동기 방식으로 결과가 표시됩니다.

## 경고
본 프로젝트는 **보안 교육 및 체험 목적**으로만 제작되었습니다. 실제 서비스 환경에 배포하는 것은 보안상 매우 위험하므로 주의하시기 바랍니다.
